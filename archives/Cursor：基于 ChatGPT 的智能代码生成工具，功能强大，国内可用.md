# Cursor：基于 ChatGPT 的智能代码生成工具，功能强大，国内可用

> 最近发现了一款名为 Cursor 的智能代码生成工具。起初，Cursor 功能较为简单，仅支持单个文件操作。但最近迎来了一次重大更新，新增了项目导入、新建、运行等功能，并提供了丰富的插件库。因此，我决定将这款工具推荐给更多人。

## 一、注册与下载

1. 访问 Cursor 的官方网站 [https://www.cursor.com/](https://www.cursor.com/)，页面设计简洁明了，点击下载按钮即可获取安装包。

![Cursor 官网界面](https://bbtdd.com/img/1523715681306613.webp)

2. 安装完成后，点击右上角的登录按钮，支持使用国内邮箱（如 QQ 邮箱、网易邮箱）注册账号。

![注册界面](https://bbtdd.com/img/763457650.webp)

## 二、功能介绍与使用指南

### 1. 代码生成与修改

1. **代码生成**：  
   在代码编辑器中，将光标移动到想要生成代码的位置，按下 `Ctrl + K`，输入代码描述即可生成对应代码。例如，输入“生成获取当前 Web 连接的系统信息和 IP 地址”，结果如下：

![代码生成示例](https://bbtdd.com/img/7678677339947.webp)

2. **代码修改**：  
   选中代码后，按下 `Ctrl + K`，输入修改需求。例如，将代码注释改为中文：

![代码修改示例](https://bbtdd.com/img/92030781467188.webp)

   修改结果如图所示：

![修改后的代码](https://bbtdd.com/img/5418373133683.webp)

   按 `Ctrl + Enter` 保存修改，或按 `Ctrl + Back` 撤销操作。

3. **代码检查**：  
   选中代码后，按下 `Ctrl + K` 检查语法错误。例如，删除方法中的字符后，工具会自动检测并修复错误：

![代码检查示例](https://bbtdd.com/img/9139183349545.webp)

4. **插件支持**：  
   Cursor 还支持通过插件运行项目，只需根据提示下载相应插件即可。

![插件支持](https://bbtdd.com/img/0770576863.webp)

### 2. 项目生成

1. **新建项目**：  
   Cursor 支持直接新建 Java 项目，操作步骤与 IntelliJ IDEA 类似。

2. **选择构建方式**：  
   点击 `JavaProjects` 上的 `+` 号，选择构建方式，如 Maven 或 Spring Boot，并配置相关参数（如 Spring Boot 版本、JDK 版本、依赖项等）。

![项目生成示例](https://bbtdd.com/img/953188877956.webp)

3. **项目结构**：  
   生成的项目层级与常规项目一致，符合开发者的使用习惯。

![项目结构](https://bbtdd.com/img/422363322.webp)
